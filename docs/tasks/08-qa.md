# Task 08 — Locales / Theme Check / a11y / responsive QA (final pass)

**Read first:** `00-context.md`. **Part A.** Depends on **all** prior tasks. Final hardening pass.

## Goal
Ship-quality pass across the whole portal: complete translations, clean Theme Check, accessibility, and cross-device responsive verification.

## Scope
- **Locales:** every string in `customer.*` present in `locales/en.default.json`; no hardcoded text anywhere; money via money filters. Add other shipped `locales/*.json` if the store is multi-locale.
- **Theme Check:** `shopify theme check` — zero errors; resolve warnings or justify.
- **A11y:** semantic landmarks, form labels + `aria-*`, focus states, keyboard nav (nav/tabs/modals/accordions), color contrast per WB style guide, ≥44px targets, `alt` text, `x-cloak` (no FOUC).
- **Responsive:** verify **375 / 750 / 1024** on every screen — dashboard (all states), orders (table↔cards), order detail, subscriptions, credits, auth, addresses, Stay.ai portal. Nav switches sidebar↔tab-bar.
- **States:** loading / empty / error for every worker-backed block; tier states FREE/VIP/ELITE; no-autoship / cancelled / empty-orders.
- **Perf sanity:** no layout shift on Alpine init; images via `picture` with correct sizes.

## Acceptance criteria
- [ ] `shopify theme check` clean.
- [ ] No hardcoded strings; locales complete.
- [ ] A11y pass (keyboard + screen-reader smoke test + contrast).
- [ ] All screens verified at 375/750/1024, all states.
- [ ] No FOUC; no console errors.
